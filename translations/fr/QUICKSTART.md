# Déploiement Rapide : Entraîner l'Architecture Cybernétique avec une Seule Phrase

Avec une seule commande, faites à l'Agent lire automatiquement le tutoriel et compléter l'entraînement complet de l'architecture cybernétique.

---

## Conditions d'Applicabilité

Votre Agent doit être capable de :
- Accéder aux dépôts GitHub publics
- Écrire dans la mémoire (fonction `memory`)
- Créer des compétences (`skill` ou modules procéduraux similaires)

Si ce n'est pas le cas, passez à [Entraînement Manuel](#entraînement-manuel).

---

## Commande en Une Seule Phrase (Copier et Envoyer à l'Agent)

```
Entraîne-moi selon scripts/deployment-commands.md de github.com/zeronesun/cybernetic-your-agent. Confirme chaque étape, puis exécute un test de boucle fermée à la fin.
```

> L'Agent lira automatiquement toutes les instructions, écrira séquentiellement le cadre de métacognition L1, initialisera L2/L3, établira la table de contrôle intégral, créera les quatre compétences fondamentales, et exécutera enfin un test de boucle fermée unique.

---

## Processus d'Entraînement (Vous n'avez qu'à Confirmer)

1. L'Agent lit le fichier d'instructions de déploiement
2. Écrit le cadre de métacognition L1
3. Initialise L2 et L3 (l'Agent vous confirme les résultats d'extraction)
4. Établit la table de contrôle intégral
5. Crée les quatre compétences : `delivery_gate`, `feedforward_startup`, `control_loop_review`, `user_profile_maintainer`
6. Exécute le premier test de boucle fermée (dites "décollage")

Le processus complet prend environ 40 minutes, vous n'avez qu'à confirmer aux étapes clés.

---

## Entraînement Manuel

Si l'Agent ne peut pas accéder à GitHub, envoyez-lui d'abord le contenu suivant :

> Je vais envoyer une liste d'instructions d'entraînement, veuillez les exécuter une par une. Informez-moi à chaque étape terminée.

Puis ouvrez 👉 [scripts/deployment-commands.md](scripts/deployment-commands.md), copiez le contenu par segments et envoyez-le à l'Agent.

---

## Indicateurs d'Entraînement Terminé

L'Agent devrait afficher :

- ✅ Les quatre compétences fondamentales ont été créées
- ✅ La table de contrôle intégral est prête
- ✅ Le premier test de boucle fermée est réussi

À ce stade, votre Agent s'est transformé en un moteur collaboratif capable d'évolution autonome, avec la cybernétique comme charpente.

---

## Aide-Mémoire d'Utilisation Quotidienne

- Nouvelle tâche → L'Agent affiche automatiquement une liste d'évitement des pièges, dites "fais direct" pour commencer
- Erreur → Lorsqu'un problème similaire apparaît pour la deuxième fois, l'Agent le transforme automatiquement en règle
- Révision after-action → Dites **"décollage"**, l'Agent exécute la révision complète en neuf étapes et s'auto-optimise
